本地离线教程

全时刻音效触发器：照着点，先让它响起来

这个教程就在本地，不用打开 GitHub。你不用理解 JSON、SoundId、路径这些词。 只要先选下面三件事之一：装别人做好的、自己做一条、或者排查为什么没响。

打开窗口：/atsound 普通用户只看 5 个标签页 社区包自动新建分组 规则用向导创建 不响先看日志

先选你要做什么

我想装别人做好的

点首页的 安装社区音效包。

进社区页后点刷新，看到喜欢的包就点安装。插件会自己新建分组。

最适合第一次体验

我想自己做一条

先去 音效库 导入 mp3/wav/ogg。

再去 我的音效 点新建规则向导，按四步走。

不用手写配置

我点了但是没响

先打开 日志。

看有没有“事件”“规则命中”“播放音效”。缺哪一段，就排查哪一段。

最快定位问题
记住一句话：下载别人的去“社区音效包”，管理自己的去“我的音效”，导入文件去“音效库”，查问题去“日志”。

第一次使用，按这个顺序来

下面这 5 步是最稳的路线。做完后，你应该能听到第一条音效。

输入 /atsound 打开插件窗口。

看到“首页、社区音效包、我的音效、音效库、日志”就对了。

去 音效库 导入一个音频文件。

选 mp3、wav、ogg 都可以。导入后可以点“测试”，先确认文件能播放。

去 我的音效，点“新建规则向导”。

新手建议先选“使用指定技能时”，技能名填你自己常用的技能。

第三步选择刚才导入的音效，第四步保存。

可以选多个音效。选多个时，触发后会随机播放其中一条。

回游戏里用这个技能，再看 日志。

日志里看到“事件到达、规则命中、播放音效”，这条链路就通了。

导入自己的音效

点 音效库，再点 导入音效文件。

选好文件后，列表里会出现这个音效。先点 测试，听得到再做规则。

先测试能不能播放

自己做一条规则

点 我的音效，再点 新建规则向导。

四步照着填：什么时候响、具体是什么、播放什么、保存到哪里。

不会填就先用默认值

让 Buff 在时循环播放

新建规则时，第一步选 获得 Buff 时。

然后勾上 Buff 在时循环播放，Buff 消失自动停止。

Buff 在，音效循环；Buff 没了，音效停。不用再额外做一条停止规则。

把音效包分享给别人

点 我的音效，打开 投稿中心。

选择要分享的内容：当前全部、某个分组，或者单条规则。

会自动带上音频和音量

安装社区音效包

点 社区音效包，点 刷新列表。

看到喜欢的包，点 安装。插件会自动新建分组，不会让你选文件夹。

装好后去 我的音效，可以停用这个分组，也可以调每个音效的音量。

社区列表来自 Gitee。断网时，插件会尽量显示上次缓存的列表和封面。

投稿到社区

投稿中心里只需要填：包名、作者、版本、封面图。

封面可以不填。没写的简介、标签、说明，插件会尽量自动生成。

生成包时，会带上你本机调好的每条音效音量。

生成后把这两个文件发送到邮箱 1104449674@qq.com，或 QQ 群 659827727。审核通过后才会出现在社区列表里。

常见问题

点了没声音，先看哪里？

先看 日志 标签页，再看音效库里的文件路径是不是还在，最后检查主音量是不是被调得太小。

为什么我填了技能名，还是所有技能都响？

通常是触发器类型选错了。请把规则的触发器选成 使用技能，不要把技能名写进动作里。

为什么两个音效一起响？

通常是有两条规则同时命中了。去 我的音效 看同一个分组或其他分组里有没有相似规则，可以先点 停用分组 快速排查。

想临时关掉一个下载包怎么办？

到 我的音效 里找到它对应的分组，点 停用分组。以后想恢复再点 启用分组。

社区包里的某个音效太大声怎么办？

到 我的音效 打开对应分组，在 这个分组的音效音量 里拖动音量条即可，也可以点测试试听。

我上传后别人下回来音量不对怎么办？

请用新版重新生成投稿包。新版分享包会带上每条音效的音量，旧版已经生成的包没有这份音量信息。

文件搬家之后不响了怎么办？

到 音效库 里重新导入，或者把条目改回新的实际路径。导入到库里以后，后面就不用自己记路径了。

GitHub 打不开也能用吗？

可以。这个教程就是本地文件，打开插件窗口里的 查看插件教程 就能看，不需要联网。
