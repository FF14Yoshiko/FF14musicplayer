# 全时刻音效触发器

作者：肘击

一个面向国服 XIVLauncherCN / Dalamud 的本地音效触发器插件。插件只使用客户端本地可见信息，监听聊天日志、战斗状态、地图切换、职业变化、HP、Buff、物品获得、击杀、本机玩家被击倒等事件，并按用户配置的分组和规则播放本地音效。

## 安装

在 Dalamud 插件安装器里添加第三方插件仓库：

```text
https://raw.githubusercontent.com/FF14Yoshiko/FF14musicplayer/main/pluginmaster.json
```

添加后搜索“全时刻音效触发器”并安装。

## 功能

- 规则触发：技能使用、物品获得、地图切换、进出战斗、HP 变化、低血量、本机玩家被击倒、职业切换、Buff 获得/消失、击杀。
- 音效库：导入 MP3/WAV/OGG，自动复制到插件管理目录；删除音效库条目时会同步清理插件导入目录里的文件。
- 首页和向导：默认只显示首页、社区音效包、我的音效、音效库、日志；新建规则用四步向导完成，不需要手写 JSON。
- 分组管理：每个分组可以一键启用/停用，下载来的音效包也可以在分组里直接调整每个音效的播放音量。
- 分享保真：导出或投稿 `.sfxpack` 时，会带上每条音效在本机设置好的播放音量，别人安装后音量一致。
- 随机音效：一条播放动作可以选择多个音效，触发时随机播放其中一条。
- 社区音效包：在插件内查看 Gitee 社区包，带封面图，点击安装后自动新建分组，并支持已安装包的更新、跳转、停用和卸载。
- 安装前检查：安装社区包前会预览内容并检测明显触发冲突，避免两个相似规则一起响。
- 社区投稿：在「我的音效」里的投稿中心选择当前内容、某个分组或单条规则，插件会做投稿预检，自动生成 `.sfxpack` 与“投稿信息.txt”，发送给作者审核后再进入社区列表。
- Buff 循环音效：获得 Buff 时循环播放，Buff 消失时自动停止。
- 高级模式：按住 Ctrl+Shift 才显示高级入口，保留完整规则编辑、分享包导入导出和审核发布。
- 内置示例：首次安装会自动提供示例音效和示例规则，包括击杀、指定技能、Buff 循环停止等常见用法。
- 本地教程：插件窗口内可直接打开离线图文教程，不依赖 GitHub 页面。

## 合规说明

插件不会自动执行任何游戏操作，不读取或显示敌方未公开数据。所有音效播放都发生在本地。

## 开发构建

需要本机已安装 XIVLauncherCN / Dalamud API 15 开发文件。

```powershell
dotnet build .\AllTimeSoundTrigger.csproj -c Release
```

构建后可发布包位于：

```text
bin\Release\AllTimeSoundTrigger\latest.zip
```
